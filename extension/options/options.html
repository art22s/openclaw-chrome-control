🦞 OpenClaw Chrome Control

Welcome! 👋

This extension lets OpenClaw AI control your Chrome browser — clicking, typing, reading pages, and more.

Let's get you set up in 3 quick steps.

Connect to Relay Server 📡

The relay server bridges the extension and OpenClaw. Make sure it's running, then test the connection.

Test Gateway Connection 🔌

Test your OpenClaw gateway connection. This goes through the relay server — no manual configuration needed.

You're All Set! 🎉

OpenClaw Chrome Control is ready. The extension will automatically connect to the relay when Chrome starts.

You can use the side panel (click the extension icon → side panel) to chat with OpenClaw, or let it control your browser directly.
